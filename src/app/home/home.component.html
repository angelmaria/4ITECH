Distintos lenguajes en programación

Java
Un lenguaje de programación de propósito general y orientado a objetos, ampliamente utilizado en el desarrollo de aplicaciones empresariales.
Angular
Un framework de desarrollo de aplicaciones web desarrollado y mantenido por Google que permite construir aplicaciones web dinámicas y de una sola página.
Python
Un lenguaje de programación interpretado y de alto nivel, conocido por su sintaxis simple y legible, ampliamente utilizado en el desarrollo web, científico y de software en general.
Node.js
Un entorno de ejecución de JavaScript del lado del servidor que permite construir aplicaciones web escalables y rápidas.
